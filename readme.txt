#################################    HOWTO    ##################################

1) read my article (all is explained inside), freely available at:
       * https://arxiv.org/abs/1407.1135
       * https://hal.archives-ouvertes.fr/hal-01018236

2) clone enhance_my_Boole (or download it if you do not use Git):
       git clone https://github.com/arnaudporet/enhance_my_Boole.git

3) open the file example.m

The work is tested with Octave version 4.0.0 under Arch Linux.

How to get Octave:
    https://www.gnu.org/software/octave/
    https://www.gnu.org/software/octave/download.html
    https://www.gnu.org/software/octave/octave.pdf

Most Linux distributions provide Octave in their official repositories.
